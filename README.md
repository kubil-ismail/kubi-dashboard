# kubi Dashboard
Kubi Dashboard is a simple landing page using react js and bootstrap

## Installation

Clone this repo

```bash
https://github.com/kubil-ismail/kubi-dashboard.git
```
Intall dependency.

```bash
cd kubi-dashboard
npm install
npm start
```
